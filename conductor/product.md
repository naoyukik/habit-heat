# Initial Concept

ここはhabit-heatという習慣管理ツールのプロジェクトです。ただのタスク管理ですが、タスクをcheckするとGitHub風草が生えるUIが用意されてます。とりあえずミニマムにそんな機能を実装したいです。

# Product Guide

## 1. Product Vision
「habit-heat」は、タスクを完了するたびにGitHub風のコントリビューショングラフが「熱を帯びる（Heat up）」視覚的体験を提供する、シンプルかつ強力な習慣管理Webツールである。

## 2. Target Audience
- **モチベーションを維持したい個人:** 日々の積み重ねを視覚的に確認し、継続の喜びを感じたいユーザー。
- **目標達成を目指す人々:** 資格勉強や運動など、特定の活動の継続を記録・確認したいユーザー。

## 3. Core Features
- **シンプルタスク管理:** タスクの作成、完了、削除。
- **Heat Graph (GitHub風草表示):** タスク完了状況をグリッド表示。GitHub風のUIを採用しつつ、「Heat」の名にちなんで継続や活動量に応じて暖色系（赤やオレンジ）に変化する視覚効果。
- **ストリークカウント:** 連続達成日数をカウントし、ユーザーの自信に繋げる。

## 4. User Experience Goals
- **ミニマリズム:** 無駄な機能を削ぎ落とし、素早くタスクを操作できるUI/UX。
- **視覚的達成感:** 「草（Heat）」が育ち、赤く染まっていく様子を楽しみながら習慣化を促進する。

## 5. Deployment & Environment
- **Platform:** Webブラウザ（マルチデバイス対応）
